# SMS-Spam-Detection-Using-ML
Machine Learning project that classifies SMS messages as Spam or Ham using TF-IDF Vectorization and Naive Bayes / Logistic Regression.
Evaluated using Accuracy, Recall, and F1-Score.   

# Technologies Used
-Python
-Pandas
-Scikit-learn
-TF-IDF Vectorizer
-Multinomial Naive Bayes / Logistic Regression

# How It Works
-Clean the text data
-Convert text into numbers using TF-IDF
-Train the model
-Evaluate using Accuracy, Precision, Recall, and F1-Score
-Predict new messages

# Model Performance
-Accuracy: 96.42%
-Recall: 99.77%
-F1-Score: 97.96%
